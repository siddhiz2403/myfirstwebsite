AutoDust-Free Smart Blackboard

Innovative classroom solution for clean, healthy teaching environments

Introduction

This system eliminates chalk dust exposure for teachers by automating board cleaning and trapping dust within a specialized compartment, contributing to better hygiene and reduced health risk.

Technologies Used

  • DC motor mechanism
  • Push button control
  • Dust-absorbing frame compartments
  • Mechanical linkage and sliders
  • Basic electronics (motor driver, switches)

How It Works

The system uses a full-width motorized duster to clean the board while capturing chalk dust using surrounding compartments.

×

Mechanism Details

When the user presses a button placed on the side of the board, a motorized duster moves horizontally across the board. As it glides, the compartments on either side absorb the chalk particles using built-in filters. The motor is calibrated to move at an optimal speed, ensuring even cleaning without scattering dust. The system was prototyped using simple mechanical elements and controlled by basic electronic components like a switch and a driver circuit.

Advantages

  • Prevents chalk dust-related respiratory issues
  • Increases teacher safety and hygiene
  • Saves time by automating board cleaning
  • Simple implementation and cost-effective

Learning & Challenges

  • Understanding dust management in enclosed systems
  • Balancing speed and torque in the motor mechanism
  • Testing materials for efficient dust trapping

Conclusion

This project offered an opportunity to explore mechanical design and address real-world classroom health concerns through a basic yet impactful automation solution.
